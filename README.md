# AT42QT
Arduino library for Atmel Touch I2C sensors

## API:
 - QT1060
 - QT1070
 - QT2120
 - QT2160
 - QT60160
 - QT60240
##### NOTES:
 - Used api from Atmel examples
 - Since 01.02.2018 - Atmel no longer exists, now it's a microchip
 
## HowTo:
 - Define your QT device in file AT42QT_API.h
 > #define QT_DEVICE QT2160
